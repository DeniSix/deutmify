# deUTMify

Simple library for cleaning up URL's from analytics shit on client side.

## Usage

```html
<script src="deutmify.js"></script>
<script>
  deUTMify.cleanup();
</script>
```

or use with module loaders.

## API

* `.cleanup()` - just do it

* `.url()` - Get clean URL
